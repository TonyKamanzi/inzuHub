# inzuHub
Find and list rental houses easily in Rwanda

# Project Description

InzuHub is a full-stack web application that allows users to browse rental houses and contact landlords directly. Landlords can list their properties, and users can search for houses based on location and price. The platform also includes an admin dashboard to manage users and listings.  This project is built with MERN stack 

## Features
- User registration & login
- Landlord dashboard to manage listings
- Tenant can search and contact landlords
- Admin dashboard to manage users and properties
- Chat system between tenants and landlords
- Dockerized backend, frontend, and MongoDB
